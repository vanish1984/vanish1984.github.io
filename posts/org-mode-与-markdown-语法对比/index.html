Org-mode 与 markdown 语法对比

Tue, Sep 8, 2020 One-minute read

整理了一下 orgmode 和 markdown 的语法对比, 有可能 Hugo 和 markdown 语法转义不是全部支持

标题

-----

 * 一级标题
 ** 二级标题
 *** 三级标题
...

-----

 # 一级标题
 ## 二级标题
 ### 三级标题
...

引用

-----

 #+BEGIN_QUOTE
 引用的文字
 #+END_QUOTE

-----

 > 引用的文字

超链接

-----

 形式1:  [[http://url][描述]]
 形式2:  [[http://url]]

-----

图片

-----

 #+CAPTION: This is the caption for the next figure link
 #+NAME:   fig:SED-HR4049
 [[link_to_image.jpg]]

-----

表格

-----

| This | is | My | First | table |
|-

<TAB>

-----

列表

-----

 * 列表项1
    + 子列表项
 * 列表项2
    1. 子列表项
 * 列表项3

 1. 列表项1
   * 子列表项
 2. 列表项2

-----

特殊格式

加粗

加粗

-----

*加粗*

-----

下划线

下划线

-----

_下划线_

-----

删除线

删除线

-----

+删除线+

-----

斜体

斜体

-----

/斜体/

-----

下标

H_2O ：水的分子式

-----

H_2O  ：水的分子式

-----

上标

E=mc^2

-----

E=mc^2

-----

行内代码

#include <stdio.h> 或者 #include <stdio.h>

-----

=#include <stdio.h>= 或者 ~#include <stdio.h>~

-----
